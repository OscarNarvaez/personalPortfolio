# Portafolio en Next.js

Portafolio personal implementado con Next.js + TypeScript.

## Requisitos

- Node.js 20+

## Uso

1. Instalar dependencias:

```bash
npm install
```

2. Levantar en desarrollo:

```bash
npm run dev
```

3. Compilar para producción:

```bash
npm run build
npm start
```

## Rutas

- `/` Home
- `/about` About me
- `/works` Works
- `/contact` Contact
